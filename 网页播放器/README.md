# 🎬 OmniPlayer Pro - 全能网页视频播放器

## 简介
OmniPlayer Pro 是一款轻量级、无需安装、纯前端运行的“瑞士军刀”级视频播放器。它不仅支持几乎所有主流视频格式，还专门针对双语视频和伴奏/人声分离的需求，提供了专业的声道控制功能。

## 核心亮点

### 📂 全格式支持
- **本地拖拽**：直接拖入 MP4, MKV, WebM 文件即可播放。
- **流媒体**：完美支持 M3U8 (HLS) 和 FLV 格式的在线直播流或点播链接。

### 🔊 专业音频控制（特色功能）
- **左右声道分离**：遇到“左声道粤语、右声道国语”或“左伴奏、右人声”的视频，可一键切换为“只听左声道”或“只听右声道”（立体声输出）。
- **多音轨切换**：支持 M3U8 内置的多语言音轨切换。

### 📝 字幕全能王
- 支持加载本地 `.vtt` 或 `.srt` 字幕文件。
- 自动识别并切换视频内置的软字幕轨道。

### 🎨 视觉与体验
- **日夜模式**：一键切换明亮/深色主题，保护视力。
- **画面滤镜**：实时调节亮度、对比度、饱和度，改善视频观感。
- **倍速播放**：0.5x ~ 2.0x 自由调节，学习刷剧两不误。

### 🔒 安全隐私
- 所有本地文件处理均在浏览器内完成，不上传任何数据到服务器，绝对保护您的隐私。

## 适用场景
- 观看下载的双语电影/港剧（声道分离）。
- 语言学习者（通过声道切换专注于单一语言）。
- 开发人员测试 M3U8/FLV 直播流。
- 不想安装庞大播放器软件的极简主义者。

## 🚀 如何使用
1. 保存代码为 `.html` 文件。
2. 双击用浏览器（推荐 Chrome/Edge）打开。
3. 将视频拖进去，开始享受！
