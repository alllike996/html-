# 🚀 个人效率仪表盘 (Personal Efficiency Dashboard)

## 简介
这是一个基于 **HTML5** 的轻量级单页应用，集成了日常工作和学习中最常用的四个小工具。  
它 **无需安装、无需联网**（加载后），所有数据仅保存在你的浏览器本地，安全且隐私。

---

## ✨ 核心功能

### 1. 🌗 沉浸式日夜模式
- **一键切换**：点击右上角的按钮，可在“明亮模式”和“深色模式”之间自由切换。  
- **智能记忆**：系统会自动记住你的偏好，下次打开时保持你上次选择的主题模式。  
- **护眼设计**：深色模式采用低对比度深灰背景，适合夜间工作，保护视力。

### 2. 📝 自动保存记事本
- **无忧写作**：输入的每一个字都会实时保存。  
- **断电保护**：即使不小心关闭浏览器窗口或刷新页面，再次打开时，你的笔记依然完好无损（基于 LocalStorage 技术）。

### 3. 📅 动态万年历
- **日期概览**：清晰展示当前月份的日期分布。  
- **今日高亮**：自动识别并高亮显示“今天”。  
- **月度切换**：支持前后翻页，查看过去或未来的日期。

### 4. 🕒 实时时钟
- **精准报时**：秒级刷新的数字时钟。  
- **完整日期**：显示包含星期几在内的完整日期信息。

### 5. 🧮 便捷计算器
- **基础运算**：支持加减乘除四则运算。  
- **可视化操作**：清晰的大按钮设计，支持退格和清空功能。
