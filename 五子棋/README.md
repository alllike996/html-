# 五子棋 · 引擎版

一个基于原生 **HTML / CSS / JavaScript** 实现的单文件五子棋游戏。  
支持 **人机对战、双人对战、引擎对战、棋谱保存/加载、复盘、AI 分析提示** 等功能，打开浏览器即可运行，无需后端或安装依赖。

---

## 项目简介

本项目是一个纯前端的五子棋网页应用，使用 `canvas` 绘制棋盘与棋子，使用原生 JavaScript 管理游戏状态、复盘逻辑和 AI 分析流程。  
整个项目可以直接以单个 HTML 文件运行，适合作为小游戏项目、前端练手项目、或原生 Web 开发示例。

---

## 功能特性

- 支持三种对战模式
  - 人机对战（PVE）
  - 双人对战（PVP）
  - 引擎对战（EVE）

- 支持多档难度
  - 简单
  - 普通
  - 困难
  - 引擎初
  - 引擎中
  - 引擎高

- 支持基础游戏操作
  - 落子
  - 悔棋
  - 重新开始
  - 显示 / 隐藏坐标

- 支持棋谱功能
  - 保存当前对局为 JSON 文件
  - 从本地 JSON 文件加载棋谱
  - 进入复盘模式查看历史步骤

- 支持复盘操作
  - 上一步
  - 下一步
  - 退出复盘

- 支持 AI 分析
  - 分析当前局面
  - 给出推荐候选点
  - 在棋盘上显示提示标记

- 支持引擎自动对战
  - 自动连续落子
  - 手动推进下一步

## 使用说明

### 1. 选择模式

左侧面板可以切换游戏模式：

- **PVE**：玩家对 AI
- **PVP**：双人本地对战
- **EVE**：双引擎自动对战

### 2. 选择难度

在人机模式下，可以选择不同难度：

- 简单
- 普通
- 困难
- 引擎初
- 引擎中
- 引擎高

### 3. 落子

点击棋盘交叉点即可落子。  
在 PVE 模式下，默认玩家执黑先手，AI 随后自动落子。

### 4. 常用操作

可通过面板进行以下操作：

- 重开一局
- 悔棋
- 切换坐标显示
- 启动 AI 分析

### 5. 保存棋谱

点击“保存”按钮，可将当前对局导出为本地 JSON 文件。

### 6. 加载棋谱

点击“加载”按钮，可从本地选择 `.json` 文件进行导入。  
导入成功后会进入复盘模式，可逐步回看历史对局。

### 7. 复盘

在复盘模式下可以：

- 查看下一步
- 回退上一步
- 退出复盘
- 对当前复盘局面进行 AI 分析

## 安全改进

本版本对棋谱导入流程做了额外安全加固，避免异常输入导致页面崩溃或状态污染。

### 已增加的保护

- 限制导入文件大小
  - 超过 1MB 的棋谱文件会被拒绝加载

- 校验 JSON 数据结构
  - 检查顶层对象是否合法
  - 检查棋盘尺寸是否匹配
  - 检查模式和难度是否在允许范围内

- 校验颜色数据
  - 玩家棋子颜色必须是合法 RGB 数组
  - 非法颜色值会回退到默认值

- 校验历史落子数据
  - 每一步必须为 `[row, col, player]`
  - 坐标必须在棋盘范围内
  - 玩家值必须合法
  - 玩家顺序必须正确
  - 不允许重复落子

- 增强复盘健壮性
  - 前进 / 后退时增加边界检查
  - 避免恶意棋谱触发运行时异常

- 释放临时资源
  - 下载棋谱后释放对象 URL
  - 页面关闭时清理 `Web Worker` 资源
