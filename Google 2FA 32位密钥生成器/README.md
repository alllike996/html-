# Google 2FA 32位密钥生成器

这是一个完全离线、单文件运行的 **2FA 密钥生成器**，基于浏览器本地存储，支持历史记录管理、二维码显示、明暗模式切换、背景图和网站图标。

---

## 功能特点

1. **随机生成 32 位 Base32 密钥**
   - 使用安全随机数生成 32 位密钥
   - 仅包含字母 A-Z 和数字 2-7
   - 可直接用于 Google Authenticator 或其他支持 TOTP 的应用

2. **二维码生成**
   - 自动生成标准 `otpauth://totp` URI
   - 可扫描二维码直接添加到验证器
   - 二维码背景为白色，模块为黑色，确保在深色背景下清晰可见

3. **历史记录**
   - 每次生成的密钥自动保存到本地历史记录
   - 历史条目显示：
     - 生成时间（含日期和时分秒）
     - Issuer 和 Account
     - 密钥后6位，前面用 `****` 隐藏
   - 历史操作：
     - **复制**：一键复制密钥
     - **加载**：加载历史条目到输入框并刷新二维码
     - **删除**：删除单条历史
     - **清空**：删除所有历史记录

4. **明暗模式切换**
   - 默认深色模式
   - 点击顶部按钮可切换到明亮模式
   - 背景图保持一致，文字颜色随模式切换

5. **页面样式**
   - 背景图：`36164226.jpg`（同目录）
   - 网站图标：`favicon.ico`（同目录）
   - 响应式布局，支持手机和桌面浏览器

6. **操作提示**
   - 所有复制操作都有 **Toast 提示**，显示 2 秒
   - 输入框修改 Issuer / Account 会实时刷新二维码

---

## 使用说明

1. **生成密钥**
   - 点击 **生成 32 位密钥** 按钮
   - 密钥会自动填入输入框并生成二维码
   - 历史记录自动更新

2. **修改 Issuer / Account**
   - 在对应输入框修改
   - 二维码会实时刷新
   - 可以重新生成二维码并保存到历史

3. **复制密钥**
   - 点击 **复制当前密钥** 按钮即可复制

4. **历史记录操作**
   - **复制**：复制该历史条目的密钥
   - **加载**：将该条目填入输入框，并刷新二维码
   - **删除**：删除单条历史
   - **清空**：清空全部历史记录

5. **切换明暗模式**
   - 点击顶部 **切换明暗模式** 按钮切换页面主题

## 安全提示

- **完全离线运行**，无需任何网络请求
- 所有数据仅保存在浏览器 **localStorage**
- 不会上传任何密钥信息，请自行备份重要密钥

---

## 技术实现

- **HTML + Tailwind CSS**（Play CDN）
- **QRCode.js**（生成二维码）
- **原生 JavaScript + Web Crypto API**（生成随机数、Base32 编码）
- **LocalStorage**（历史记录存储）
- **单文件运行，响应式设计**
